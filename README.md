## Getting Started

  

### Installation

1. Clone the repo

```sh

git clone git@bitbucket.org:mygedev/new.rent.myauto.ge.git

```

3. Install NPM packages

```sh

yarn / npm install

```

4. Add in project  `src/env.js`

```js

module.exports  ={
	API_URL: '',
	API_HOSTNAME: '',
	ACCESS_TOKEN_NAME: 'token',
	TOKEN_TIME_MINUTES: 600,
	TOKEN_EXPIRE_TIME_NAME: 'trtn',
	NODE_ENV: 'dev'
}

```

5. Run project

```sh

yarn dev / npm run dev

```
